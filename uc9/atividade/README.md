# 📱 App Cardápio

Aplicativo mobile desenvolvido com **React Native + Expo + TypeScript**, com foco em aprendizado prático de desenvolvimento mobile.

O projeto consiste em uma tela simples de cadastro de produtos/pratos, sendo a base para evolução ao longo da disciplina.

## 🚀 Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- Node.js (ambiente de execução)

---

## 📦 Como executar o projeto do zero

### ✅ Pré-requisitos

Antes de começar, você precisa ter instalado:

- Node.js (versão 18 ou superior)
- npm ou yarn
- Expo CLI (opcional, mas recomendado)

---

### 🔧 Passo a passo

```bash
# 1. Criar o projeto com Expo
npx create-expo-app@latest app-cardapio

# 2. Acessar a pasta
cd app-cardapio

# 3. Instalar dependências
npm install

# 4. Iniciar o projeto
npm start

# Clonar o repositório
git clone https://github.com/2025-08-53/app_cardapio.git

# Entrar na pasta
cd app_cardapio

# Instalar dependências
npm install

# Rodar o projeto
npm start
